# Развертывание frontend модуля:
**1.** Создать каталог, где будут располагаться файлы frontend модуля:
```
  ..\OpenServer\domains\module\frontend 
```
Каталог с module проекта должен быть уже создан. (*См. инструкцию по развертыванию backend модуля.*)

**2.** Скачать файлы frontend модуля по адресу в виде zip-архива:
```
  https://github.com/gird-it/module-frontend
```

**3.** Содержимое zip-архива распаковать в каталог, созданного в **п.1.** (*именно содержимое zip-архива, без создания нового каталога*)

**4.** Среди распакованных файлов найти файл .env.example, создать на его основе файл .env

**5.** Открыть файл *.env* и отредактировать HOST в соответствии с BASE_URL в *.env* файле backend модуля (*См. инструкцию по развертыванию backend модуля.*).

Backend модуль:
```
  BASE_URL=http://module.gird.ru
```
Frontend модуль:
```
  HOST=http://module.gird.ru
```

**6.** Запустить ряд npm команд в консоли:
+ Загрузить пакеты frontend модуля с помощью команды (займет некоторое время):
```
  npm install // После команды должна появится папка node_modules.
```
+ Запустить сборку для разработки приложения:
```
  npm run dev
```
После сборки, в терминале будет показан адрес сборки.

**7.** В браузере перейти по адресу:
```
  http://module.gird.ru:9000/
```
